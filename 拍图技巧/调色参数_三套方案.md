# 调色参数速查 · 三套方案

> 手机修图调色参数，存着直接用

---

## 🎨 方案一：反差色滤镜（富士滤镜风）

| 参数 | 数值 |
|------|:--:|
| 滤镜 | 反差色 / 富士滤镜 |
| 曝光 | -10 |
| 鲜明度 | +8 |
| 高光 | +20 |
| 阴影 | -10 |
| 亮度 | +6 |
| 自然饱和度 | -7 |
| 色温 | -15 |
| 晕影 | +10 |

> 效果：冷调、有质感、带暗角，适合氛围图

---

## 🎨 方案二：原相机清透感

| 参数 | 数值 |
|------|:--:|
| 鲜明度 | +20 |
| 高光 | -10 |
| 阴影 | +25 |
| 对比度 | -15 |
| 自然饱和度 | -5 |
| 色温 | -10 |

> 效果：柔和清透、冷白皮感，适合人像/毛衣平铺

---

## 🎨 方案三：原相机高级暗调

| 参数 | 数值 |
|------|:--:|
| 曝光 | -25 |
| 照明度/亮度 | +15 |
| 高光 | -20 |
| 阴影 | -15 |
| 对比度 | +7 |
| 亮度 | +6 |
| 锐度 | +20 |

> 效果：暗调高级感、纹理清晰，适合细节/面料展示
