# 明天傻瓜式操作指南（一步一步跟着做）

---

## ⏰ 上午第一步：准备东西（10分钟）

### 先找这三样：

**① 一面白墙**
- 家里找一面干净的白色墙壁
- 旁边要有插座（后面手机可能要充电）
- 别在窗户正对面，侧面对着窗户最好

**② 一张桌子**
- 书桌、餐桌、办公桌都行
- 最好是白桌子，不是白的话铺块白布
- 别要亮面反光的（玻璃桌不行）

**③ 一个全身镜**
- 能照到全身的穿衣镜
- 没有的话，对着落地窗玻璃反光也能拍
- 拍照前用纸巾擦干净，尤其拍深色衣服

### 小道具随便找几个：
- 家里翻翻：咖啡杯、杂志、干花、毛线团
- 楼下捡：落叶、树枝、松果
- 实在没有就不放，也行

---

## ⏰ 上午第二步：调手机（2分钟）

打开手机相机 → 跟着做：

### 设置一：调成正方形
```
1. 打开相机
2. 屏幕上方向左/右滑，找到「1:1」或「正方形」
3. 选中它
```
> 看见画面变正方形就对了 ✅

### 设置二：打开九宫格
```
1. 相机界面点右上角「设置」⚙️（或三条杠）
2. 找到「网格」或「九宫格」
3. 打开开关
```
> 屏幕上出现4条交叉线就对了 ✅

### 设置三：切换到视频模式
```
1. 相机底部选「录像」或「视频」
2. 屏幕顶部确认是「4K 60fps」（看不到就算了）
```
> 底部红点是录像模式 ✅

---

## ⏰ 上午第三步：拍第一款（7分钟）

> 先拿一件样衣试，别紧张，拍不好就重来

### 🌑 拍法一：关灯拍氛围图（2分钟）

```
1. 走到选好的白墙边
2. 把房间灯全关了（台灯也别开）
3. 窗帘拉一半，让一点点自然光进来
4. 把衣服随意丢在地上/桌上
5. 旁边放一杯咖啡或一片落叶
6. 手机举在衣服正上方，用九宫格把衣服放左上角
7. 别把衣服全拍进去，留一点悬念
8. 按快门
```
> ✅ 检验：画面暗暗的，衣服和道具看得见，有氛围感

### 🪑 拍法二：白桌拍细节（3分钟）

```
1. 把衣服随意丢在桌子上
2. 不要刻意整理，让它自然皱一点
3. 搭配个眼镜盒/包包/杂志在旁边

先拍全景：
4. 手机举在正上方，衣服占画面2/3
5. 别全拍进去，留一点桌边
6. 按快门

再拍细节：
7. 手机靠近衣服，拍领口纹理，对焦点在毛线上
8. 按快门
9. 再拍袖口/下摆，手指轻捏展示弹性
10. 按快门
```
> ✅ 检验：全景1张 + 细节2张 = 共3张

### 🪞 拍法三：对镜自拍（2分钟）

```
1. 穿上这款样衣
2. 站到全身镜前面，距离镜面大约一步
3. 手机拿在手里，屏幕对着自己
4. 点开相机 → 选「视频」模式
5. 按红点开始录
6. 手机往下半身靠近镜子（遮住脸）
7. 侧一点身，脚往前伸一点
8. 停几秒，换一个姿势再停几秒
9. 录大概30秒就够了
10. 再按红点停止
```
> ✅ 检验：视频里脸被手机挡住，衣服看得清，姿势自然

**第一款拍完了！同样的方法拍第二、三、四款。**

## ⏰ 下午第四步：照片修一修（每款2分钟）

### 打开醒图：

```
1. 打开醒图APP
2. 点「导入」→ 选刚才拍的平铺图（最正那张）
3. 点底部「调节」
4. 找到这些选项，一个个调：

   亮度 → 往右滑到 +5
   锐化 → 往右滑到 +10
   对比度 → 往左滑到 -3
   色温 → 往左滑到 -2

5. 点右上角「保存」
```
> ⚠️ 别点滤镜！别点美颜！别调饱和度！

---

## ⏰ 下午第五步：做AI模特（每款3分钟）

### 打开美图设计室：

```
1. 打开美图设计室APP
2. 首页找「AI商拍」或「AI模特」→ 点进去
3. 点「上传服装图」→ 选修好的平铺图
4. 等AI自动抠图（几秒钟）

检查抠图：
5. 看领口/袖口边缘干不干净
6. 不干净 → 点「手动修正」→ 用橡皮擦掉多余的
7. 干净了就点下一步

选模特：
8. 模特选「亚洲」「韩系」标签
9. 姿势别选站姿！选「微侧身」「行走」「撩发」
10. 场景选「咖啡厅」或「街拍」或「浅灰背景」
11. 比例选 1:1

生成：
12. 点「生成」
13. 等10-30秒
14. 不满意就再点生成（同一设置生成3次，挑最好的）
```
> ✅ 挑图标准：衣服没变形 + 颜色接近实物 + 姿势不僵硬

### 把AI模特图保存到相册。

---

## ⏰ 下午第六步：做四宫格拼图（每款3分钟）

### 打开美图秀秀：

```
1. 打开美图秀秀APP
2. 首页搜「拼图」→ 点进去
3. 选4张图：
   - AI模特图
   - 平铺图
   - 细节图（领口或面料）
   - 氛围素材图（搜"灰色系氛围图"截图就行）
4. 选模板「四宫格」
5. 调成上下对角排版（不是四个等分）
6. 点完成
```

### 加水印：

```
1. 美图秀秀底部找「文字」
2. 输入你的品牌名「Lucky」
3. 字体选最细最简单的（无衬线）
4. 颜色选白色
5. 字号调小，放在角落
6. 透明度调到30-40%（隐隐约约看得到就行）
7. 保存
```
