# 不露脸对镜自拍 · 四宫格教程

## 前期准备
- 镜子：1000元以上重镜（轻薄镜易进灰发糊），拍照前擦干净
- 手机：苹果14+ → 原相机「视频模式」4K 60帧 HDR（别用拍照，会锐化）
- 光线：自然光

## 拍照技巧
- **显脸小**：手机靠近镜子，脸被镜子自然挡住
- **显腿长**：靠墙站，脚往前伸一丢丢
- **构图**：距离镜子1米，脚贴画面底部，头顶留白露眼，不全拍

## 后期
1. 剪映：导入视频 → 剪废镜 → 配BGM关原声 → 手动美体拉腿 → 截帧保存
2. 美图秀秀：搜"XX色系氛围图" → 穿搭图+细节图+氛围素材 → 四宫格对角拼图 → 加英文水印

## 成品
穿搭图 + 细节图 + 氛围感素材，对角排版，不露脸突出质感，2分钟一套。
