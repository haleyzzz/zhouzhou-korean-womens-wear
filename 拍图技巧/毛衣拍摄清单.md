# 毛衣拍摄 + AI模特 全流程

> 适用：TEMPL/玄 韩系毛衣 | 工具：手机 + 醒图 + 美图设计室

---

## 📋 准备清单

| 物品 | 用途 |
|------|------|
| 白色大卡纸/白布 | 色卡对比背景 |
| 深灰/燕麦色背景布 | 平铺底色（深色衣用浅底，浅色衣用深底） |
| 自然光窗边 | 光线（上午10点/下午3点） |
| 手机三脚架（没有也行） | 防手抖 |

---

## 🔆 第一步：布光（2分钟）

1. 找到家里**采光最好的窗户**
2. 拉一层**白色薄窗帘**或挂白布柔光
3. 拍摄台放窗边**侧面45°**，不要正对窗户
4. 关掉所有室内灯（灯光会混色偏黄）
5. ✅ 检验：手放在拍摄台上，影子柔和、无重影 = 合格

---

## 📸 第二步：拍6张平铺图（5分钟/款）

> ⚠️ 手机设置：开**2倍变焦**，关掉AI美颜/HDR，开网格辅助线

### 图① — 正面平铺

```
操作：
1. 衣服铺平，领口自然展开，袖子45°摆放
2. 手机与衣服平行（别俯拍），站正上方
3. 衣服占画面的80%，四周留白
4. 对焦点在胸口位置
5. 拍完检查：领口左右对称吗？
```

### 图② — 领口特写

```
操作：
1. 手机靠近到15-20cm
2. 对焦在领口织法纹理上（手点屏幕锁定）
3. 拍出线圈结构
4. 角度：斜面30°，不要正俯拍
```

### 图③ — 袖口/下摆特写

```
操作：
1. 一只手轻捏袖口，展示弹性
2. 另一手持手机对焦
3. 拍出螺纹收口细节
4. 同样拍下摆
```

### 图④ — 面料纹理（微距）

```
操作：
1. 手机调微距模式（或开到最近对焦距离）
2. 对准衣服最显毛感的位置
3. 拍出纱线的蓬松感/线圈感
```

### 图⑤ — 色卡对比

```
操作：
1. 衣服旁边放一张A4白纸
2. 白纸和衣服并排入镜
3. 目的在于：后期可参照白纸校准颜色
4. 白纸不能过曝
```

### 图⑥ — 叠穿氛围（可选）

```
操作：
1. 这款毛衣叠搭一件基础款（白T/衬衫领）
2. 拍胸以上的局部
3. 目的：给氛围感，不是展示全貌
```

---

## 🖼️ 第三步：醒图修图（3分钟/款）

> 目标：让图更清晰，不是改变颜色

| 步骤 | 操作 | 数值 |
|------|------|------|
| 1 | 打开醒图 → 导入图① | — |
| 2 | 调节 → 亮度 | +5 |
| 3 | 调节 → 锐化 | +10 |
| 4 | 调节 → 对比度 | -3 |
| 5 | 调节 → 色温 | -2（去黄） |
| 6 | 裁剪 → 3:4（小红书）/ 1:1（朋友圈） | — |
| 7 | 保存原图（不要压缩） | — |
| 8 | 图②-⑥同样批量处理 | — |

> ⚠️ 别加滤镜！别调饱和度！毛衣颜色必须真实

---

## 🤖 第四步：AI模特生成（3分钟/款）

### 4.1 选图

```
从6张里挑「图①正面平铺」，
这是唯一用来生成AI模特的图。
（其他5张是细节图，直接发圈用）
```

### 4.2 美图设计室操作

```
1. 打开美图设计室 → 首页「AI商拍」
2. 点「上传服装图」→ 选修好的图①
3. AI自动抠图 → 检查边缘：
   - 领口/袖口边缘是否干净？
   - 不干净→点「手动修正」用橡皮/恢复笔微调
4. 进入「选择模特」页面
```

### 4.3 参数设置（关键！）

| 设置 | 怎么选 |
|------|--------|
| **模特** | 亚洲/韩系标签 → 自然感优先 |
| **姿势** | 不选站姿！选「微侧身」「行走」「撩发」 |
| **场景** | 「咖啡厅」「街拍」「浅灰背景」 |
| **比例** | 1:1（朋友圈）/ 3:4（小红书） |
| **生成** | 同设置生成3张，挑最好的 |

### 4.4 挑图标准

```
✅ 我满意：
- 衣服版型没变形
- 颜色和实物接近
- 模特姿势自然不僵硬

❌ 重来：
- 衣服颜色跑偏严重
- 纹理完全糊掉
- 模特手指变形/眼神诡异
```

## 📱 第五步：发圈/发小红书

### 朋友圈（1:1）

```
第1张：AI模特上身（吸引眼球）
第2张：领口特写
第3张：面料纹理
第4张：正面平铺原图
```

### 小红书（3:4）

```
第1张：AI模特上身（封面！！！）
第2张：正面平铺原图
第3张：领口特写
第4张：面料微距
第5张：色卡对比
第6张：叠穿氛围/袖口细节
第7-9张：同款其他颜色平铺
```
