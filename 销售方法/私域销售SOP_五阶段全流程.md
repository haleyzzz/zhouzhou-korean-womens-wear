# 私域销售SOP·五阶段全流程

> 参哥商学院

---

## 第一阶段：起盘准备期

### 1. 同行调研
- 全网搜同行账号，加同行微信
- 拆解：产品线、销售方式、朋友圈框架、如何升单（二转三转）
- 调研市场：行业现状分析 + 未来预测

### 2. 分析人设树
人物小传 → 企业内核 → 核心优势 → 成长史 → 主要成就 → 行业热词 → 经典案例

### 3. 提炼优势点
人设背景 → 人生经历 → 从业时长 → 行业成就 → 产品卖点 → 对比同行优势

### 4. 梳理产品线+海报
人设简介 | 适合人群 | 实力展现 | 过往结果 | 好评案例 | 产品特色

### 5. 微信注册设计
头像 + 名字 + 朋友圈背景 + 地点 + 个性签名 → **核心：像真人一样使用**

### 6. 养号+登记表
- 日常消费是必须动作
- 新微信每天进人 ≤20人
- 主动添加客户 ≤5人

### 7. 打造朋友圈（每天≥3条）
| 类型 | 内容 |
|------|------|
| 我是谁 | 身份、日常生活、人生观、价值观 |
| 我有什么 | 产品 + 结果 |
| 为什么找我 | 客户反馈和好评 |

### 8. 设计客户标签（动态调整）
```
A：已成交      B：高意向/差点付费
C：沟通未付费  D：初步了解
E：刚添加      F：防封内部人员
```
格式：沟通情况 + 时间 + 年级 + 成交产品

### 9. 分析人群痛点
人群分类 → 人群分析 → 痛点摘取 → 提炼需求

### 10. 制定销售路径
了解客户 → 破冰共情 → 分析挖掘需求 → 打痛点 → 给画面给希望 → 引导付费 → 解决异议成交

### 11. 复盘转化情况
- **未转化**：分析卡点 → 找原因 → 沟通方案
- **已转化**：分析成交原因 → 分享思路 → 复制路径

### 12. 整理一阶SOP
人设介绍（优势+产品+体系） → 适合人群 → 人群痛点 → 销售路径

## 第二阶段：项目熟悉期

### 业务跟进
- 每天1个成功案例 + 1个失败案例
- 整理好评库 + 客户踩坑库 + 购买话术库

### 营销调整
朋友圈迭代架构：
1. 好评案例（每天收集）
2. 成交案例（出单截图）
3. 视频号（专业+产品）
4. 人设内容（观点+日常生活）

### 产品调整
- 产品整合打包
- 思考方向：产品周期、产品特性

---

## 第三阶段：项目瓶颈期

### 数据分析出问题

| 问题 | 分析 | 解决 |
|------|------|------|
| 转化问题 | 业务能力/客户跟进 | 深度培训、精选案例、收集异议、重分类人群、精细化维护 |
| 流量问题 | 数据+节点沟通+复盘 | 阶段会议+单聊+人生规划 |
| 人员状态 | — | 待定 |

---

## 第四阶段：迭代上升期

1. 再次分析同行/跨行
2. 分析数据：转化、未转化、粉丝人群分类
3. 私域营销：朋友圈从3条→5条，增加互动
4. 员工动力：日流量数据、销售卡点、好做法、求助
5. 工作节点把控

---

## 第五阶段：稳定复制期

### 数据分析
每日流量加人（数量+人群）+ 每日转化（出单+客单价）

### 流量反馈
反馈客户需求 = 找爆款内容方向的最简单方式

### 团队状态
文化灌输 + 晋升机制 + 目标树立 + 阶段奖励

### 优秀动作标准化复制
1. 收集优质聊天记录
2. 收集好评和异议
3. 日常工作互动+回访
4. 每日工作总结
5. 阶段性分析同行
6. 私域直播
7. 节点式沟通+周期复盘
8. 制作里程碑
