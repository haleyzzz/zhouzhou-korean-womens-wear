流程名称：测款发小红书
使用场景：新款样衣出来后验证市场需求
输入来源：样衣实物照片
处理方式：(1) 拍照 6~9 张（平铺/细节/上身）(2) 加水印 @TEMPL 玄 (3) 小红书发笔记，标题+正文+5个话题标签 (4) 不定位（防同行+三四线城市不精准）

# 8. 判断是否应该写入 memory 的标准
如果一条信息满足以下任意条件，可以写入 memory：
* 未来会反复用到
* 已经被多次确认
* 会影响项目推进

* 是重要项目的长期事实

如果不满足，就不要写入 memory。
