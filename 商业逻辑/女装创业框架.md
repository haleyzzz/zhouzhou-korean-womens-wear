# 女装创业 · 科班系统框架

> 核心：不碎片化摸索，搭建完整闭环

---

## 第一层：地基认知

| 模块 | 要点 |
|------|------|
| 商业基础 | 流量、转化、产品、复购。盈利=差价/服务费/会员 |
| 市场认知 | 客户买单=解决痛点+满足欲望+降低焦虑。只做刚需 |
| 风险底线 | **禁止大额囤货、租高价门店。先验证订单再投成本** |

---

## 第二层：赛道选型

**优选：** 本地实体 / 垂直细分 / 技能服务（零货源）

**避雷：** 红海大众款 / 大额加盟 / 靠人脉的生意

**自我匹配三步：** 兴趣 → 资源 → 启动资金

---

## 第三层：四大核心能力

| 能力 | 关键 |
|------|------|
| 获客 | 公域种草 + 私域留存 + 老客转介绍 |
| 销售 | 挖需求 → 打消顾虑 → 分层成交 |
| 产品 | 对标爆款优化 → 小批量测款 → 差异化 |
| 财务 | 公私账户分开，不负债扩张 |

---

## 第四层：落地执行三步走

| 阶段 | 时间 | 动作 |
|------|:--:|------|
| 最小闭环 | 7-15天 | 不租店不囤货，朋友圈上样品收预定 |
| 小范围放大 | — | 订单稳定后少量备货，沉淀老客户 |
| 系统化放大 | — | 标准化流程，拓展多渠道 |

---

## 第五层：复盘迭代

- 周度：哪个渠道客户多？哪款好卖？哪步流失多？
- 月度：砍低效、推爆款、学同行新玩法

---

## 新手禁忌 🚫

1. 重资产开店囤货
2. 只学不落地
3. 频繁换赛道
4. 只顾拉新不复购
