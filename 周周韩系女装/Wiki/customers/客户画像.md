# 客户画像

> 你的典型客户是谁？他们来自哪里？怎么选货？

## 核心客户类型

### 类型一：实体服装店主

| 维度 | 描述 |
|------|------|
| 身份 | 各地服装零售店主 |
| 来源 | 城市/县城/乡镇（待细化） |
| 进货频率 | 旺季频繁，淡季稀疏 |
| 选货方式 | 到店翻款看图挑货 |
| 需求 | 好卖的、有利润的、符合当地审美的 |

### 老客户特征

| 维度 | 描述 |
|------|------|
| 互动深度 | 不仅挑货，会聊市场、聊趋势 |
| 信息价值 | 高——是免费的市场调研员 |
| 当前问题 | 聊完不记录，信息流失严重 |
