# Canva 封面设计

> 小红书笔记封面图的制作流程

## 核心工具

- **Canva（可画）**：免费手机 App，模板丰富
- **尺寸**：3:4 竖版（1080×1440 像素）

## 设计原则

| 原则 | 说明 |
|------|------|
| 大字清晰 | 标题 8-12 字，手机小屏看得清 |
| 配色统一 | 选定品牌色系（如奶白+棕+深灰），不超过 3 色 |
| 产品突出 | 毛衣图片占封面 50%+ 面积 |
| 标题有吸引力 | 封面标题负责吸引点击 |

## 操作流程

1. 打开 Canva → 搜索"小红书封面"模板
2. 选韩系风格模板 → 替换文字和图片
3. 用「复制此设计」保持统一风格，只改文字图片
4. 导出 PNG/JPG 到相册
